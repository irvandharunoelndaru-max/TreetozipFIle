Paste a directory tree – press Download ZIP to get the exact folder structure.

📄 .zip
💡 How it works:
  • Lines ending with / become folders.
  • Other lines become empty files (you add content later).
  • Indentation is read from the visual position – tree characters (├──, │) are handled automatically.
  • Comments like (project-level) are removed.
